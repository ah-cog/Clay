Setting up BOM for P1R5
Use existing parts as a template for new ones.
Add part information from the vendor/manufacturer website as shown in earlier entries to 
    work with bom-ex.
Put path to the database in a global attribute on your schematic called "DATABASE"
Run bom-ex.ulp. RTFM from there. jeez. (pdf of manual is in the bom-ex download.)
